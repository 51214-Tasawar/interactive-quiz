1. Which keyword is used to declare a variable in JavaScript?
A. var
B. let
C. const
D. All of the above
Correct: D

2. What is the output of: typeof null?

A. "null"
B. "object"
C. "undefined"
D. "boolean"
Correct: B

3. Which method converts JSON data to a JavaScript object?

A. JSON.parse()
B. JSON.stringify()
C. parse.JSON()
D. JSON.convert()
Correct: A

4. Which symbol is used for single-line comments in JavaScript?

A. #
B. //

D. **

A. function = myFunction()
